Google Apps Script 주소를 연결하면 PC·모바일 데이터가 자동 동기화됩니다.

1
구글 시트 → 확장 프로그램 → Apps Script → AppScript.gs 붙여넣기 → 저장
2
배포 → 새 배포 → 유형: 웹 앱 → 액세스: 모든 사용자 → 배포 → URL 복사
3
복사한 URL을 아래 입력
Apps Script 웹 앱 URL
올바른 URL을 입력해주세요.
🔒 URL은 이 브라우저에만 저장됩니다. 모바일에서도 같은 URL을 입력하면 데이터 공유.

🔄 동기화 설정 (최초 1회)
1
구글 시트 만들기
sheets.new → 빈 시트 생성
2
Apps Script 열기
시트 상단 확장 프로그램 → Apps Script
3
코드 교체 후 배포
AppScript.gs 내용 붙여넣기 → 저장 → 배포 → 새 배포 → 웹 앱 → 모든 사용자 → URL 복사
4
URL 연결
상단 🔗 URL 변경 → 복사한 URL 입력. 모바일에서도 동일 URL 입력 시 데이터 공유.
데이터는 시트 A1(일정) · B1(할일) 셀에 자동 저장됩니다.
액세스 권한을 반드시 모든 사용자로 설정해야 합니다.
❓ 동기화 실패 시
↻ 동기화 재시도 → 계속 실패 시 Apps Script 재배포 후 🔗 URL 변경으로 새 URL 입력.
🌐 GitHub Pages 배포 방법
1
GitHub 저장소 만들기
github.com/new → 저장소 이름 입력 → Public → Create repository
2
파일 업로드
저장소 페이지 → uploading an existing file → SemoView-Daily.html 드래그 앤 드롭 → Commit changes
3
Pages 설정
저장소 Settings → Pages → Branch: main → /(root) → Save
4
접속 주소 확인
https://유저명.github.io/저장소명/SemoView-Daily.html 형태로 접속 가능. 배포까지 1~2분 소요.
파일 업데이트 시: 저장소에서 기존 파일 클릭 → ✏️ 편집 아이콘 → 파일 내용 교체 → Commit changes
📱 모바일 설정 (홈 화면 추가)
iPhone Safari: 하단 공유 버튼(□↑) → 홈 화면에 추가
Android Chrome: 우측 상단 ⋮ → 앱 설치 또는 홈 화면에 추가
홈 화면에 추가하면 앱처럼 전체화면으로 실행됩니다.
